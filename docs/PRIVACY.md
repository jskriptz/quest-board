# Privacy Policy

**Last Updated:** April 8, 2026

## Overview

Quest Board is a client-side productivity application. This policy explains how your data is handled.

## Data Collection

### What We Collect
- **None directly.** Quest Board runs entirely in your browser.

### Local Storage
The app uses browser localStorage to save:
- Character stats and progress
- Quest lists and completion status
- Uploaded portrait images (base64 encoded)
- Equipment and inventory data
- Your Anthropic API key (if you choose to enable AI features)

This data never leaves your device except:
- Your API key is sent to Anthropic's API when generating quest text
- Quest descriptions you enter are sent to Anthropic for AI processing (if AI enabled)

## Third-Party Services

### Anthropic API (Optional)
AI quest generation is **optional** and requires you to provide your own API key.

**If you enable AI features:**
- Your API key is stored in your browser's localStorage
- Your API key is sent directly to Anthropic's servers (nowhere else)
- The task description you enter is sent to Anthropic for processing

**Data NOT sent to Anthropic:**
- Character stats
- Progress or XP
- Portrait images
- Any other personal information

**If you don't enable AI:** No data is sent anywhere. The app works fully offline.

Anthropic's use of data is governed by [Anthropic's Privacy Policy](https://www.anthropic.com/privacy).

## Data Storage

- All game data is stored locally in your browser
- No server-side storage exists
- No accounts or registration required
- No cookies used for tracking

## Data Deletion

To delete all Quest Board data:
1. Clear your browser's localStorage for this site
2. Or use your browser's "Clear site data" feature

## Children's Privacy

Quest Board does not knowingly collect data from children under 13. The app has no user accounts or registration.

## Changes to This Policy

Updates will be posted to this page with a revised "Last Updated" date.
